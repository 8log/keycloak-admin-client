Module: users

Methods

<inner> create(realmName, user)

A function to create a new user for a realm.
Parameters:
Name Type Description
realmName string The name of the realm(not the realmID) - ex: master
user object The JSON representation of a user - http://keycloak.github.io/docs/rest-api/index.html#_userrepresentation - username must be unique
Returns:
A promise that will resolve with the user object
Type
Promise
Example
keycloakAdminClient(settings)
    .then((client) => {
      client.users.create(realmName, user)
        .then((createdUser) => {
        console.log(createdUser) // [{...}]
      });
    });

<inner> find(realmName, options)

A function to get the list of users or a user for a realm.
Parameters:
Name Type Argument Description
realmName string The name of the realm(not the realmID) - ex: master
options object <optional>
 The options object
Properties
Name Type Argument Description
userId string <optional>
 use this options to get a user by an id. If this value is populated, it overrides the querystring param options
username string <optional>
 the querystring param to search based on username
firstName string <optional>
 the querystring param to search based on firstName
lastName string <optional>
 the querystring param to search based on lastName
email string <optional>
 the querystring param to search based on email
Returns:
A promise that will resolve with an Array of user objects or just the 1 user object if userId is used
Type
Promise
Example
keycloakAdminClient(settings)
    .then((client) => {
      client.users.find(realmName)
        .then((userList) => {
        console.log(userList) // [{...},{...}, ...]
      });
    });
